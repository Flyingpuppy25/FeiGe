## 💡 Key Features / 核心功能

*   **🎬 1键拆解 (One-Click Deconstruction)**：支持影视、短视频、广告视频的一键智能拆片与分镜识别。
*   **🔬 4步风格研究 (4-Step Style Research)**：采用「准备 → 分镜拆解 → 构建拼图 → AI 提炼」完整四阶段流程，快速解构视觉风格。
*   **🖼️ 镜头拼图 (Shot Collage)**：自动选择最多 12 个代表镜头构建拼图，交由 AI 提炼可复用的视觉风格。
*   **🔒 本地安全 (Local & Private)**：API Key 在本机填入与加密保存，不经过任何第三方服务器，不含在任何导出包中。
